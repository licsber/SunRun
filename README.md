# SunRun
本repo为阳光长跑的Go实现  
实话说 实现原理已经烂大街了
适用于阳光长跑  
阳光体育服务平台

# 抓包
## 安卓端
使用`packet capture`抓取登录包即可  
打开主界面之后的一般为第一个包  

### 一般长这样

```
{
  "Success": true,
  "Data": {
    "Token": "0d3bf9578beb4*******a9384272a488",

    "AndroidVer": 2.1*,
    "AppleVer": 1.2*,
    "WinVer": 1.0

}
```

## ios端
目前好像接口失效了  
参见魔盒运动的一个神奇方法(bundle id)  
自行搜索吧 以前用的抓包应用叫`stream`

# 使用方法
`git clone git@github.com:Licsber/SunRun.git`  
`cd SunRun`  
`vim main.go`  
在这里调用`run.go`的`justRun`方法即可  
参数`imeicode`为抓包获取  
参数`distance`为学校要求  
一般男生2400 女生2000即可
